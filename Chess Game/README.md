Chess Game Engine

Este proyecto es una implementación básica de un juego de ajedrez desarrollada en Python utilizando Pygame. Su propósito principal es trabajar sobre la lógica del juego y la gestión del estado, más que ofrecer una experiencia completa o competitiva.

El programa mantiene internamente el estado del tablero, controla los turnos, registra los movimientos realizados y permite deshacer la última jugada. Cada acción del usuario se traduce en cambios claros y consistentes sobre ese estado.

La interfaz gráfica se limita a mostrar el tablero, las piezas y a capturar la interacción del usuario. Toda la lógica del juego está separada de la capa visual, lo que permite entender el proyecto como un pequeño motor de reglas conectado a una interfaz simple.

Este proyecto sirve como base para explorar conceptos como modelado de estado, manejo de eventos y diseño de lógica de dominio, y puede extenderse fácilmente con validación de movimientos, inteligencia artificial o una arquitectura más avanzada.

Tecnologías utilizadas: Python 3 y Pygame.
